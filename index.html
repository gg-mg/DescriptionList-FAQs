Sample Description List

TCP
Transmission Control Protocol ia a method (protocol) used along with the Internet Protocol (IP)
to send data in form of message units, called packets, between computers over the internet.
IP
Internet Protocol is the method or protocol by which data is sent from one computer to another
on the Internet. Each computer on the internet is uniquely identified by an IP address.
FTP
File Transfer Protocol is a protocol used to exchange files between computers on the Internet.
HTTP
Hypertext Transfer Protocol is the protocol used for exchanging text, graphic images, sound, video,
and other multimedia files on the Web
